## Что это такое

Образовательный тренажёр для взрослых людей, которые хотят разобраться в истории России без скуки и без ощущения, что снова сидят в школе. Идея в том, что история — это не набор дат для зубрёжки, а живые события с причинами и последствиями.

Проект называется **Хроники России**.

## Технологии

- Python 3.10+
- Django 5.x
- SQLite (база данных)
- HTML + CSS (Playfair Display, PT Serif) + JavaScript
- Без сторонних frontend-фреймворков
