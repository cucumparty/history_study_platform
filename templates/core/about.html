О проекте

Тренажёр для взрослых, которые хотят знать историю

Зачем это нужно

Многие взрослые люди хотят знать историю своей страны лучше, но традиционные учебники кажутся скучными, а курсы — слишком академичными. Этот проект создан для тех, кто хочет учиться в своём темпе, без давления и экзаменов.

Как устроен проект

  • 📜 Лекции — материал разбит на небольшие читаемые блоки с тестом в конце.
  • 🃏 Карточки — запоминай даты и события с помощью флипкарточек.
  • 🏆 Квизы — проверяй себя по любой теме в режиме игры.
  • 💡 Факты — неожиданные подробности, которых нет в учебниках.
  • 🌍 Таймлайн — видь, что происходило в России и в мире одновременно.

Технический стек

Проект написан на Python / Django, данные хранятся в SQLite. Фронтенд — чистый HTML + CSS + немного JS, без тяжёлых фреймворков.
